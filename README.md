# 🚀 AutoReq

<div align="center">

### Plataforma para Gestão e Validação de Requisitos de Software

Solução web desenvolvida para centralizar o levantamento, organização, validação e acompanhamento de requisitos, promovendo maior colaboração entre clientes, analistas e desenvolvedores.

</div>

---

## 📖 Sobre o Projeto

O **AutoReq** foi criado para solucionar problemas recorrentes no processo de engenharia de requisitos, como falhas de comunicação, documentação descentralizada, perda de informações e retrabalho durante o desenvolvimento de software.

A plataforma oferece um ambiente colaborativo onde equipes podem gerenciar projetos, cadastrar requisitos, validar informações e gerar documentação de forma automatizada, garantindo maior rastreabilidade e organização ao longo do ciclo de vida do projeto.

---

## 🎯 Objetivos

* Centralizar informações em um único ambiente.
* Melhorar a comunicação entre stakeholders e equipe técnica.
* Facilitar o gerenciamento de requisitos.
* Reduzir retrabalho causado por falhas de entendimento.
* Automatizar a geração de documentação.
* Aumentar a produtividade e a organização dos projetos.

---

## ✨ Funcionalidades

### 📂 Gestão de Projetos

* Criação, edição e exclusão de projetos.
* Organização dos requisitos por projeto.

### 📝 Gerenciamento de Requisitos

* Cadastro de requisitos funcionais e não funcionais.
* Definição de prioridade.
* Atualização e acompanhamento do status.

### 👥 Colaboração

* Comentários e interações entre usuários.
* Histórico de alterações.
* Validação e revisão de requisitos.

### 🔒 Segurança

* Sistema de autenticação.
* Controle de acesso por usuário.

### 📄 Documentação

* Anexar documentos.

### 📊 Acompanhamento

* Dashboard para visualização de requisitos e andamento dos projetos.
